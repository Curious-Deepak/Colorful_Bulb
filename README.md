# ColorFull_Bulb

**ColorFul Glowing Bulb** a modern glowing bulb UI animation created using **pure HTML and CSS**, enhanced with a beautiful color picker powered by **iro.js** (via CDN).

This project demonstrates how far CSS alone can be pushed to create realistic lighting effects, modern UI components, and elegant animations — without relying on JavaScript frameworks.

## Author Of This Project 

- Deepak
- Project Type : Portfolio Project


## Purpose of This Project

The main goals are to :

- Practice advanced CSS techniques.
- Explore glow, blur, and realistic lighting effects using only CSS.
- Design modern UI components with minimal JavaScript.
- Learn how powerful CSS variables can be for theming and dynamic visuals.
- Create a clean, portfolio-ready UI experiment.

Perfect for beginners and intermediate frontend developers who want to level up their CSS skills !

## Features

- Pure HTML & CSS bulb design
- Realistic glowing light with large-radius glow effect
- Smooth pulse animation for natural lighting
- Modern glass-style (frosted glass) settings panel
- Elegant custom toggle switch (styled with pure CSS)
- Dynamic color selection using **iro.js** color picker

## Technologies Used

- **HTML5**
- **CSS3**
  - CSS Custom Properties (Variables)
  - Flexbox
  - Gradients
  - Box-shadow & backdrop-filter (blur)
  - Pseudo-elements (`::before`, `::after`)
  - Keyframe animations
- **iro.js** (color picker library loaded via CDN)

```html
<script src="https://cdn.jsdelivr.net/npm/@jaames/iro@5"></script>
```
